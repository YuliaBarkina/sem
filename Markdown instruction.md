## Списки 

Чтобы создать ненумерованные списки необходимо пункты выделить (*). Например, вот так:

* Элемент 1

* Элемент 2

* Элемент 3

Чтобы создать нумерованные списки необходимо пункты пронумеровать. Например, вот так:

1. Пункт

2. Пункт

3. Пункт
